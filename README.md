# monthly
Monthly Dividend App  
주식현황을 입력 받고 해당 주식들의 배당정보를 제공하고 주식정보 시각화를 제공하는 서비스

# 개발 기간
2020.07.06~2021.01.09

# 사용 기술
mobile - Flutter  
back - Flask
